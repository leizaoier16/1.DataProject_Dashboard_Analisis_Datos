# 1.DataProject_Dashboard_AnalisisDatos

Análisis de ventas y precios de vehiculos: Limpieza, Exploración y Dashboard 

DESCRIPCION DEL PROYECTO
Este proyecto realiza un ánalisis completo del conjunto de datos de automóviles proveniente de Kaggle.
El objetivo es transfromar, limpiar y explorar la información para comprender los factores que influyen en la venta y precios de los vehículos, también asi intentando identificar patrones en función de variables como la marca, año y  kilometraje de los vehículos, por ejemplo.

Se emplearon técnicas de:
Limpieza y transformación de datos.
Análisis descriptivo.
Visualización interactiva mediante un dashboard dinámico en Excel.
Elaboración de un informe explicativo con conclusiones finales.

RESULTADOS Y CONCLUSIONES

El análisis del conjunto de datos muestra que el mercado es bastante homogéneo, con diferencias pequeñas entre marcas en cuanto a ventas y precios. Aun así, se observan algunas tendencias útiles:

Marcas: No hay grandes variaciones, aunque Toyota destaca ligeramente tanto en ventas como en precio medio.
Kilometraje y año: No presentan patrones claros; el precio y las ventas se distribuyen de forma uniforme.
Combustible: Sí muestra diferencias relevantes; algunas marcas venden más diésel, otras gasolina o híbridos.
Transmisión: La automática tiene un precio medio un poco superior, pero la manual se vende ligeramente más.
Cilindrada: Los coches de 3–4 L son los más caros, mientras que los más vendidos están entre 2–3 L.

En general, no existen factores que generen cambios drásticos en el precio o las ventas, pero estas pequeñas tendencias pueden ayudar a compradores y vendedores a tomar mejores decisiones.
Por ejemplo, optar por marcas como Toyota o por vehículos diésel puede alinearse con lo que más se vende según el dataset.
